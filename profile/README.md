## Overview

Data Guardian Advanced is a secure personal data management application that stores passwords, PINs, credit card numbers, software licenses, and sensitive notes in an AES-encrypted database accessible only through your master password.

The encryption happens entirely on your device with no cloud synchronization unless you choose it, keeping your most sensitive information under local control without depending on the security practices of a third-party service.

Custom field templates create structured record types for different categories of sensitive information, letting you define exactly what fields each type of record contains rather than forcing all data into a generic password form.

Fast search across all records surfaces any stored information immediately, making Data Guardian Advanced useful as a reference tool during daily work rather than just a place to deposit credentials and forget them.

For users who want a locally-stored, strongly encrypted personal database for organizing sensitive information without the subscription model or cloud dependency of mainstream password managers, Data Guardian Advanced provides the protection and organization they need.

## Performance

Data Guardian balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management reduces overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance compared to software-only solutions. On Intel machines, hardware acceleration is used wherever available.
